Welcome to Online BookStore Database

Welcome to this simple micro web framework! With just a few lines of code, you can retrieve information about an online bookstore. This framework is designed to be fast and easy to use, making it the perfect solution for those who need quick access to bookstore data.

Tech Stack

This online bookstore website is built using Python Flask micro web framework, with a sleek and modern design powered by Bootstrap. We utilize Amazon Web Service Relational Database System (RDS) with SQL Server for the backend connection to our bookstore database, ensuring fast and reliable access to the latest releases, bestsellers, and classics.

The web framework communicates with the backend cloud service seamlessly using the Pyodbc library. This library provides a reliable and efficient connection between the web framework and the cloud service, allowing for smooth data transfer and retrieval.
